This is the Repo for 262P-Weekly Exercises

Java version used: 21.0.6

- To run the code of WEEK3, use the following command:

  - For Seven.java:

    ```
    cd week3
    ```

    ```
    javac Seven.java
    ```

    ```
    java Seven pride-and-prejudice.txt
    ```

  - For Eight.java:
    ```
    cd week3
    ```
    ```
    javac Eight.java
    ```
    ```
    java -Xss32m Eight pride-and-prejudice.txt
    ```
